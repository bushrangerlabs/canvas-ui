# Canvas UI - Visual Dashboard Editor

Transform your Home Assistant into a professional automation hub with Canvas UI's drag-and-drop dashboard builder.

## What's New in v2.0

- ✨ **AI View Generator** - Create complete dashboards from text prompts
- 🎨 **20+ Widgets** - Expanded widget library with weather, calendar, and keyboard widgets
- 🎯 **Enhanced Editor** - Professional alignment tools, grid snapping, and visual guides
- 📱 **Kiosk Mode** - Full-screen display mode for tablets and wall panels
- 🔧 **React Architecture** - Modern TypeScript/React foundation for better performance

## Key Features

### Visual Editor

- Drag-and-drop widget placement
- Multi-select and group operations
- Alignment and distribution tools
- Grid snapping and guides
- Undo/redo support

### AI-Powered

- Natural language view generation
- Automatic widget configuration
- Smart entity suggestions
- Lovelace card discovery

### Professional Widgets

- Advanced gauges with custom zones
- Interactive controls (sliders, knobs, switches)
- Data visualization (charts, progress indicators)
- Media widgets (camera feeds, images)
- Custom icon support (17,000+ icons)

### Kiosk Mode

Perfect for dedicated displays:

- Full-screen presentation
- Hide HA interface elements
- Direct URL access to specific views
- Optimized for touch interaction

## Installation

After installation via HACS:

1. Add to your `configuration.yaml`:
   ```yaml
   canvas_ui:
   ```
2. Restart Home Assistant
3. Canvas UI will appear in your sidebar
4. Start creating beautiful dashboards!

## Getting Started

1. **Create a View** - Click "+" to start a new dashboard
2. **Add Widgets** - Drag from widget library onto canvas
3. **Configure** - Select widgets and edit properties
4. **Bind Data** - Connect widgets to your entities
5. **Preview** - Switch to view mode to see it live
